# Policies

This document records privacy, safety, and usage policies for Echo AI.

## Data Handling
- Store only synthetic or fully de-identified data.
- Never commit audio recordings or raw sensitive inputs.
- Apply schema validation to all structured payloads.

## Safety and Review
- Echo AI is not a diagnostic system.
- All outputs require human review and approval.
- Outputs must be traceable and reviewable.

## Logging and Retention
- Log only minimal metadata required for auditing.
- Enforce deletion or retention policies as configured.
- Avoid storing model inputs that include sensitive data.

## Access and Security
- Limit access to authorized contributors only.
- Keep API keys and secrets out of the repo.
